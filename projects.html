Note Keeper

This is a React application that allows users to save and delete notes. The goal of this project was to practice creating React components, managing state with hooks (useState), using map/filter/reduce functions, conditional rendering using ternary and && operators, array destructuring, and using spread operators while also adding additional styling to a complete React application. View it on Glitch.

Emoji Journal Prompts

A React application that uses the OpenAI API to generate journal prompts based on emojis the user selected. Technologies include React, Express, Node, JavaScript. Further implementations will include a MongoDb backend to save journal entries with AI providing a weekly summary of entries and mood statistics.

Todo List

Simple todo list application using MongoDB, Express, Node, and Javascript. Users can add and remove list items, as well as create custom lists by appending "/" and list name to the URL (for example /Work).

Stickman 2D Platformer Game

Stickman is a 2D platformer game where players control Stickman to progress through multiple levels by avoiding or killing enemy slimes, picking up coins, and reaching the chest at the end of each level! Built with Java, JavaFX and Gradle.
